# Next.js QR Code PDF Generator

## Run locally

1. Install dependencies:

```
npm install
```

2. Start dev server:

```
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push this project to GitHub or GitLab.
2. Import into Vercel.
3. It will detect Next.js automatically and deploy.
